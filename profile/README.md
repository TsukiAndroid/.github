## 🌟 What is Tsuki Manga Reader?

**Tsuki** (月) means "moon" in Japanese — and just like the moon, Tsuki is always there when you need it.

Tsuki Manga Reader is a powerful, free Android app for reading manga, manhwa, and manhua. Unlike other manga readers that break when sites go down or developers abandon them, Tsuki is built from the ground up to keep working — no matter what.

Whether you're on a brand new flagship or an old Android 6 device, Tsuki works for you.

## ✨ Why Choose Tsuki App?

### 📚 1200+ Built-in Sources
Powered by kotatsu-parsers-redo, Tsuki ships with over 1200 manga, manhwa, and manhua sources out of the box. Always updated, always working.

### 🔍 Universal Manga Detection
Tsuki can detect and add ANY manga website as a source automatically — just browse the site and Tsuki figures out the rest. No setup, no coding, no CSS selectors.

### 🧠 AI-Powered Parser Learning
Open any manga site in Tsuki's built-in browser. Tsuki silently watches your browsing and automatically generates a working parser — zero effort from you.

### 🌐 Browser Sources
Add any manga website as a Browser Source. Browse and read from any site with full reading history, progress tracking, and native reader integration.

### 🎯 Visual Rule Builder
If a source breaks, tap "Fix Source Visually" — Tsuki loads the website and you simply tap on the manga title, cover, and chapter list. Tsuki generates the parser automatically from your taps.

### 📱 Android 6+ Support
Most manga readers dropped support for older devices. Tsuki was literally born because of an Android 7 device that couldn't run anything else. No device left behind.

### 🛡️ Ad Blocker Built-in
Tsuki's browser has a built-in ad blocker specifically tuned for manga sites. No more popup ads, redirects, or crypto miners.

### 🎨 Material You UI
Dynamic color theming on Android 12+, AMOLED dark mode, and a clean modern interface built with Jetpack Compose.

### ♾️ Built for Immortality
Multiple layers of immortality — built-in sources, AI parser learning, browser sources, universal detection, DNS over HTTPS, and Cloudflare bypass. Tsuki never truly breaks.

## 📖 About

Tsuki was created by a solo developer who couldn't find a manga reader that worked on their Android 7 device. After trying every app available, they decided to build one that would never break — for themselves and for everyone else in the same situation.

Tsuki is completely free, open source, and will remain that way forever. This is a vibe codded project so expect some bugs.

**Built with:** Kotlin · Jetpack Compose · kotatsu-parsers-redo · QuickJS · OkHttp

**Supported platforms:** Android 6.0 and above (arm64, arm32, x86_64, universal)
